This post demonstrates the built-in MDX rendering mechanism.

## Why MDX

MDX combines Markdown with JSX, so you can write expressive technical articles and still embed components when needed.

## LaTeX Support

Inline math works like this: $E = mc^2$.

Block equations work too:

$$
\mathrm{softmax}(z_i)=\frac{e^{z_i}}{\sum_{j=1}^{K}e^{z_j}}
$$

## Code Highlighting

```ts
export function costPerToken(totalCostUsd: number, tokens: number) {
  if (tokens <= 0) return 0
  return totalCostUsd / tokens
}

console.log(costPerToken(12.5, 250000))
```
